### FVM管理flutter版本

FVM（Flutter Version Manager） 是 Flutter 的版本管理器，用于：
•   安装多个 Flutter 版本
•   在不同项目中使用不同版本
•   解决 Flutter 升级导致项目无法构建的问题

### flutter常用的命令
1、flutter常用的指令

```
查看flutter的版本
fvm flutter --version
安装flutter 的版本
fvm install 2.8.1
安装flutter 开发环境的检查
fvm flutter doctor
强制切换flutter SDK 的版本
fvm use 2.8.1 --force
移除 flutter的对应的版本
fvm remove 2.8.1
清理缓存
fvm flutter clean
安装依赖包
fvm flutter pub get
查看当前可运行的设备
fvm flutter devices
查看设备
flutter devices
运行指定到模拟器
flutter run -d emulator-5556 

```
2、通过命令行启动安卓模拟器（无需打开 Android Studio）

```
1、打开终端（Terminal）。
2、进入 Android SDK 的 emulator 目录：
cd ~/Library/Android/sdk/emulator   # macOS
或者 Windows
cd %ANDROID_HOME%\emulator
3、列出可用的虚拟设备：
./emulator -list-avds   # macOS/Linux
emulator -list-avds     # Windows
4、启动指定虚拟设备：
./emulator -avd <AVD_NAME>
例如：
./emulator -avd Medium_Phone_API_36.1

```
3、通过指令打开iOS模拟器

```
open -a Simulator
```
